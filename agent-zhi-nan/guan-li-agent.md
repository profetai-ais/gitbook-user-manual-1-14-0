# 管理 Agent

## **Agent 的種類**

<figure><img src="../.gitbook/assets/image (136).png" alt=""><figcaption></figcaption></figure>

* **聊天 Agent** ：透過基礎問答直接與內部或外部大語言模型互動。
* **工作流程**：可編排工作流程處理複雜的應用場景。

> 注意：「聊天 Agent 」不能搜尋企業內部資訊生成回覆，若要使用企業內部知識請使用「知識」型 Agent 。

## **指派 Agent 庫管理人員**

<figure><img src="../.gitbook/assets/image (137).png" alt=""><figcaption></figcaption></figure>

<figure><img src="../.gitbook/assets/image (138).png" alt=""><figcaption></figcaption></figure>

1. 點擊畫面靠左上方的 「 … ( 更多 ) 」按鈕
2. 選擇 「指派」
3. 點擊彈出視窗中的 「使用者選單」選取使用者，或是輸入文字篩選
4. 點擊\*\*「 Agent 庫角色選單」選取要賦予使用者的角色
5. 點擊「邀請」按鈕完成權限分配
6. 點擊使用者列最右側的「移除」按鈕便可刪除賦予給使用者的權限

> 若要變更使用者分配的角色，點擊使用者列中的「 Agent 庫角色名稱」便可重新選擇

## 匯出 / 匯入 Agent （JSON）

匯出 / 匯入功能可用於備份 Agent 設定、在不同環境間搬移設定，或快速複製相似 Agent 以降低重複設定成本。

### 匯出 Agent

<figure><img src="../.gitbook/assets/image (139).png" alt=""><figcaption></figcaption></figure>

1. 前往 Agent 清單，或進入目標 Agent 的編輯頁面。
2. 滑鼠移動到需匯出的 Agent 表格上。
3. 選擇 匯出（Export）按鈕，系統會下載一個 proas 檔案。

### 匯入 Agent

<figure><img src="../.gitbook/assets/image (140).png" alt=""><figcaption></figcaption></figure>

<figure><img src="../.gitbook/assets/image (141).png" alt=""><figcaption></figcaption></figure>

1. 點擊畫面靠左上方的 「 … ( 更多 ) 」按鈕
2. 選擇「匯入檔案」
3. 點擊欄位選擇匯入的檔案
4. 點擊「確定」匯入指定的檔案
