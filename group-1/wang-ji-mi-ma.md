# 忘記密碼

## 已設定 Mail Server 時重設密碼 <a href="#yi-she-ding-mail-server-shi-zhong-she-mi-ma" id="yi-she-ding-mail-server-shi-zhong-she-mi-ma"></a>

當系統已設定 Mail Server 時，使用者可以透過密碼重設郵件自行重新設定密碼。​

<figure><img src="../.gitbook/assets/image (297).png" alt=""><figcaption></figcaption></figure>

<figure><img src="../.gitbook/assets/image (298).png" alt=""><figcaption></figcaption></figure>

<figure><img src="../.gitbook/assets/image (299).png" alt=""><figcaption></figcaption></figure>

<figure><img src="../.gitbook/assets/image (300).png" alt=""><figcaption></figcaption></figure>



1. 在登入頁面點擊 **Forgot Password?**。
2. 輸入您的帳號資訊。
3. 點擊 **Retrieve Password**。
4. 系統會發送密碼重設郵件至該帳號綁定的電子信箱，開啟郵件後，點擊郵件中的密碼重設按鈕。
5. 系統會導向密碼重設頁面，輸入並確認您的新密碼。
6. 密碼更新完成後，請回到登入頁面，使用新密碼重新登入。

> 注意 : 請確認輸入的帳號資訊與已註冊的帳號相符。若未收到密碼重設郵件，請先檢查垃圾郵件或促銷郵件資料夾，或聯繫您的管理員取得協助。

​

## 未設定 Mail Server 時重設密碼 <a href="#wei-she-ding-mail-server-shi-zhong-she-mi-ma" id="wei-she-ding-mail-server-shi-zhong-she-mi-ma"></a>

當系統未設定 Mail Server 時，系統無法發送密碼重設郵件。使用者需要聯繫管理員，請管理員協助重設密碼。

<figure><img src="../.gitbook/assets/image (297).png" alt=""><figcaption></figcaption></figure>

<figure><img src="../.gitbook/assets/image (301).png" alt=""><figcaption></figcaption></figure>

<figure><img src="../.gitbook/assets/image (302).png" alt=""><figcaption></figcaption></figure>

<figure><img src="../.gitbook/assets/image (303).png" alt=""><figcaption></figcaption></figure>

<figure><img src="../.gitbook/assets/image (304).png" alt=""><figcaption></figcaption></figure>



1. 在登入頁面點擊 **Forgot Password?**。
2. 系統會顯示提示視窗，告知您需要聯繫管理員重設密碼。
3. 請自行聯繫管理員，並提供需要重設密碼的帳號資訊，管理員會將臨時密碼提供給使用者。
4. 回到登入頁面，輸入您的帳號與臨時密碼。
5. 使用臨時密碼登入後，系統會導向密碼重新設定頁面，輸入並確認您的新密碼。
6. 密碼更新完成後，後續請使用新密碼登入系統。

> 注意 : 基於安全考量，使用臨時密碼登入後，請立即完成密碼重新設定。請勿將臨時密碼或新密碼提供給他人。
