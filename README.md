## **核心功能與技術特點**

* **Knowledge Base**：簡單三步驟建立文件知識庫，解決大語言模型內建知識不足與不可靠回覆問題，並能授權領域專家管理並在安全的情況下維護各自知識庫，避免關鍵知識外洩。
* **Agent**：快速依據不同應用場景產生 Agent，並透過授權讓使用者快速應用。
* **提示詞應用 / 模板管理**：固化常用的提示詞並將其 app 化，讓使用者簡單提供關鍵資訊提高問答的精確性，避免需要自己輸入大量提示詞。
* **AI 工作流程 (AI Workflow)**：彈性的 AI 流程機制，解決不同 RAG 應用情境與可整合外部整合搜尋機制，提高 RAG 設計效率，讓 AI Studio 的應用提升更多可能性。
